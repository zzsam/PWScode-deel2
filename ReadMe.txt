Ons Programma start je op door GUIpws te runnen.

Vanuit hier kan je via help --> tutorial te weten komen hoe de rekenmachine zelf werkt.

Hierin staat alleen geen uitleg over de standaardrekenmachine.

Deze werkt als volgt :
Je kan maar met 2 cijfers werken, daarna moet je op C drukken.
Kies je cijfer en druk op een van de volgende +,-,/,* en dan op =
Daarna zie je je antwoord. De andere knoppen werken nog niet en het klopt dat de cijfer tijdens het intypen nog niet te zie zijn. Het is nog work in progress.
